Menu barra de herramientas

Hay dos vistas diferentes disponibles para los iconos de barra de herramientas: nuevos y clásicos. Cada uno de los iconos visibles en la barra de herramientas proporciona consejos emergentes. Simplemente mueva el puntero del ratón sobre el icono y aparecerá un pequeño consejo. Para mayor comodidad, se proporciona una referencia rápida para cada uno de los iconos.

Puedes personalizar (a partir de la versión 3.11) los iconos que aparecen en la barra de herramientas y el orden en que aparecen.

Cambiar entre las vistas de iconos

  1. Ir al menú Ver
  2. Seleccionar Barra de herramientas clásica o Nueva barras de herramientas
