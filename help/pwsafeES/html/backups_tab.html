Pestaña copias de respaldo

La pestaña de copias de respaldo te permite configurar como se crean y manejan los copias de seguridad automáticas.

Guardar base de datos inmediatamente tras Editar o Añadir

Estando seleccionado, Password Safe guarda todos los cambios hechos en una entrada automáticamente, sin esperar a que se use el menú Guardar o Guardar como y sin confirmación.

Crear archivos de respaldo intermedios (.bak) antes de guardar

Estando seleccionado, Password Safe copiará la base de datos antes de cada vez que se guarde. Los nombres y ubicación de las copias de respaldo se definen en los siguentes campos, al igual que cuantas copias deben guardarse.

Nombre de copia de respaldo

Los nombres de los archivos de respaldo consisten en Base (por defecto el nombre de la base de datos sin la extensión .psafe3), y Sufijo (nada, un contador o la fecha y hora de la copia de respaldo). Aquí puedes configurar como se nombra al archivo de respaldo, y cuantos se grabarán (Password Safe borrará automáticamente los más antiguos tras el número de archivos de respaldo que indiques). Esto sólo funciona si el sufijo es un número incremental (un contador).

Directorio de respaldo

Por defecto, Password Safe almacena las copias de respaldo en la misma carpeta que la base de datos abierta. En este campo puedes indicar una localización alternativa, como una carpeta especial para copias de seguridad, un disco de red, un llave estraíble, etc. Por supuesto, debes tener permiso de escritura en la ubicación que indiques.
