Actualizar desde versiones anteriores de Password Safe

A fin de soportar nuevas funciones y mejorar la seguridad, el formato de base de datos que se utiliza para almacenar las entradas y preferencias se ha cambiado entre las versiones principales. Sin embargo, se ha puesto mucho esfuerzo en garantizar una actualización automática y transparente:
  • La versión 1.x de Password Safe sólo puede leer bases de datos generadas por las versiones 1.x.
  • La versión 2.x de Password Safe puede leer bases de datos creadas por las versiones 1.x y 2.x de Password Safe
  • La versión 3.x de Password Safe puede leer bases de datos creadas por todas las versiones previas.
En todos los casos, para leer una base de datos generados por una versión anterior de Password Safe, sólo ábrela como cualquier base de datos. Las entradas se mostrarán con normalidad, con posibles valores "desconocido" de los campos que no estaban soportados en versiones anteriores (por ejemplo, la fecha de creación de una entrada).
Al guardar una base de datos de una versión anterior, se le alertará del hecho de que la base de datos guardada será ilegible para la versión anterior de Password Safe.

Generación de versiones anteriores de la base de datos

Puedes crear versiones de la base de datos que puedan ser leídos por las versiones más antiguas de Password Safe exportandola explícitamente al formato deseado (Archivo > Exportar a y seleccionar formato v1.x... or formato v2.x...). Tenga en cuenta que las bases de datos exportadas no contienen información que fue añadida en las versiones más recientes. Por ejemplo, una base de datos de la versión 2 no dispondrá de información relativa a la fecha de modificación de entradas.
