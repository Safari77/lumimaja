Fijar una política de contraseñas

La politica de contraseñas describe las reglas que se usan para crear una contraseñas—las letras, mayúsculas o minúsculas, longitud y cuaquier carater especial. En algunos casos sólo se requieren letras minúsculas en otros se solicita una letra mayúscula, numeros, o caracteres especiales, así como especificar la longitud mínima. En general, es una buena idea usar la máxima longitud permitida y mezcla de caracteres de todo tipo.

El diálogo de política de contraseñas te permite fijar la política general de generación de contraseñas. Esta politica puede modificarse temporalmente y las contraseñas pueden editarse en cualquier momento en el dialogo Editar/ver entrada.

Seleccionar reglas de contraseñas

Cada elemento "usar" (usar números, usar simbolos, etc.) seleccionado en el diálogo fuerza a que Password Safe incluya los caracteres especificados en las contraseñas que genera. Cada contraseña se generará con la longitud especificada (ni más corto ni más largo). La única excepción es la opción Usar sólo caracteres hexadecimales. Si está seleccionada, las contraseñas incluirán sólo caracteres hexadecimales. Seleccionar esta opción desactiva todas las demás casillas de verificación.

Seleccionando Generar contraseñas pronunciables hará que Password Safe genere una palabra "sinsentido" que debe ser pronunciable, tal como "ingstria", "ringsony", etc. Esto es útil cuando necesitas una contraseña que sea tanto segura como relativamente fácil de recordar sin Password Safe. Eligiendo esta opción user números y/o usar símbolos permitirá reemplazar algunas letras con sus equivalentes, por ejemplo, "E" se reemplaza con "3", "S" con "5", etc.

Los caracteres que se añaden cuando se selecciona usar símbolos son : + - = _ @ # $ % ^ & ; : , . < > / ~ \ [ ] ( ) { } ? ! | a menos que también esté seleccionado usar sólo caracteres de fácil lectura, en cuyo caso los caracteres son sólo: + - = _ @ # $ % ^ & < > / ~ \ ?

Las letras que se usan cuando está marcadao usar sólo caracteres de fácil lectura son: a b c d e f g h i j k m n o p q r s t u v w x y z y A B C D E F G H J K L M N P Q R T U V W X Y, y los números se limitan a 3 4 6 7 8 9.
