Menú archivo

El menú archivo contiene opciones relacionadas con el manejo y creación de nuevas bases de datos de Password Safe.

Nuevo

Crea una nueva base de datos de contraseñas.

Abrir

Abre una base de datos de contraseñas existente.

Cerrar

Cierra la base de datos abierta sin salir de la aplicación.

Recientes

Esta sección del menú archivo lista las bases de datos abiertas recientemente. Esta sección estará vacía si es la primera vez que se usa Password Safe.

Notas:

  • La lista de archivos abiertos recientemente puede limpiarse seleccionando "Limpiar archivos recientes"
  • El número de archivos recientes que mostrará Password Safe puede configurarse en el diálogo Herramientas->Opciones->Sistema, donde también puedes elegir si prefieres ver los archivos recientes en el menú archivo, o como un submenú.

Guardar

Guardar cualquier cambio en la caja de seguridad actual. Si no se han hecho cambios desde la anterior vez que se guardó, o si la base de datos se abrió como sólo lectura, esta opción estará desactivada.

Guardar como

Guarda una copia de la caja de seguridad actual con un nuevo nombre o ubicación.

Exportar a

Exporta el contenido de la base de datos actual a un archivo externo en un formato de base de datos anterior de Password Safe, XML, o como texto. Aquí tienes más información sobre importar y exportar la base de datos a otros formatos.

Importar de

Importa el contenido de una base de datos exportada anteriormente o un archivo de texto correctamente formateado, XML, o una base de datos KeePass, en la base de datos actual. Aquí tienes más información sobre importar archivos de texto.

Combinar

Combina una base de datos Password Safe o un archivo que copia de seguridad (.bak) con la base de datos actual. Desabilitado en modo sólo lectura.

Comparar

Compara la base de datos abierta con otra.

Sincronizar

Sincroniza la base de datos abierta con otra.

La principal diferencia entre Combinar y Sincronizar es que durante la combinación, todos los elementos de la base de datos que se mezcla se llevan a la base de datos abierta. En una sincronización, sólo los elementos de la base de datos abierta se actualizarán.

Nota: Las operacione de combinar, comparar y sincronizar pueden llevar mucho tiempo, especialmente en bases de datos grandes.

Propiedades

Muestra información de la base de datos abierta, como: versión del formato, número de grupos, número de entradas e información sobre cuándo y quién realizó la última grabación.

Salir

Cierra el programa Password Safe. Si se hicieron cambios desde la última vez que se guardó la base de datos, se te preguntará si quieres guardarlos.
