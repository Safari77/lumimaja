Pestaña Misc


Varios valores por defecto del programa se muestran en la pestaña Misc.

Confirmar borrado de elementos

Estando seleccionada, se mostrará un mensaje de confirmación cada vez que se intente borrar una entrada de la base de datos.

Registro de últimos accesos

Estando seleccionada, Password Safe registra la hora en que se accede a cada entrada (cuando se copió la contraseñas o cuando se vieron los detalles de la entrada, etc.). La hora se almacena con la entrada y puede consultarse en el diálogo de la entrada.
Notas:
  • Estando habilitado, la base de datos se modificará sin que se edite una entrada. Esto significa que puedes ser advertido de grabar la base de datos antes de salir, incluso sin haber cambiado nada explicitamente.
  • Cuando la base de datos se abre en modo sólo lectura, las horas de acceso no se actualizan.

La tecla Escape cierra la aplicación

Estando seleccionada, puede usarse Esc para cerrar Password Safe cuando el programa está maximizado. Si está activado el icono en la bandeja de sistema (a través de la opción Mostrar icono en el área de notificación), entonces Esc sólo cierra la ventana principal, que puede reabrirse haciendo doble clic en el icono del área de notificación.

Tecla rápida

Estando selecionada, presionando las tecla en la secuencia a la derecha restaurará PasswordSafe desde el área de notificación, si está minimizado, o lo traerá al primer plano, si está tras otras aplicaciones. La secuencia puede cambiarse cuando la opción está seleccionada.

Acción al hacer doble clic

Te permite definir qué acción hará PasswordSafe cuando hagas doble clic en una entrada. El valor por defecto es copiar la contraseña de la entrada al portapapeles.  Las opciones disponibles son:

  • Autoescritura
  • Abrir URL
  • Ir a URL + Autoescritura
  • Copiar notas al portapapeles
  • Copiar contraseña al portapapeles
  • Copiar contraseña al portapapeles, minimizar
  • Copiar usuario al portapapeles
  • Ejecutar comando
  • Enviar correo
  • Ver/Editar entrada seleccionada

El elemento copiar contraseña y minimizar no funcionará como se espera si está seleccionada la opción "Borrar portapapeles al minimizar" en la pestaña de opciones de Seguridad.

Autoescritura

  • Minimizar tras autoescritura Por defecto, Password Safe se minimiza al realizar autoescritura. Si la opción no está marcada, Password Safe se situará tras todas las ventanas mostradas en su lugar.
  • Formato autoescritura por defecto Aquí puedes indicar la cadena a usar por defecto para todas las entradas. Puedes modificar este comportamiento en cada entrada. Esta opción es útil, por ejemplo, si quieres cambiar los retardos para todas las entradas. Consulta aquí para más detalles sobre autoescritura.

Usuario por defecto

Esto te permite fijar un usuario por defecto que se utilizará automáticamente cada vez que se genere una nueva entrada. Estando seleccionado, el nombre introducido se insertará por defecto en el campo usuario de las nuevas entradas. Por supuesto, puedes editarlos si es necesario.

Navegador alternativo

El navegador que indiques en este campo  se usará al seleccionar la opción "Abrir URL" de una entrada, cuando la URL de la entrada tenga "[alt]" antes de la dirección, por ejemplo, "[alt]http://www.foo.net". Esto es útil, por ejemplo, si tu navegador por defecto es Firefox y quieres acceder a un sitio triste y lamentable que funciona mejor con Internet Explorer, en cuyo caso debes indicar Internet Explorer como tu Navegador alternativo (normalmente "C:\Archivos de programa\Internet Explorer\iexplorer.exe).
Puedes indicar parámetros al navegador a tras del campo Parámetros de línea de comandos.
