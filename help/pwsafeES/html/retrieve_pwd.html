Usar contraseñas almacenadas

Password Safe proporciona varios mecanismos para utilizar nombres y contraseñas almacenados. Se puede copiar el nombre de usuario o contraseña al portapapeles y luego pegar la información en el campo de entrada requerido. Password Safe proporciona una función, Autoescritura, que automatiza la introducción de usuario y contraseña en un formulario web.

  1. Abre Password Safe
  2. Selecciona la entrada con el usuario/contraseña que quieres usar
    Nombre de usuario:
  • Selecciona el icono de usuario de la barra de herramientas
    o
  • Clic derecho y selecciona Copiar usuario al portapapeles
    o
  • Usa Ctrl + U
    y depués Pega el nombre de usuario en el campo deseado.
    Contraseña:
  • Selecciona el icono de contraseña de la barra de herramientas
    o
  • Clic derecho y selecciona Copiar contraseña al portapapeles
    o
  • Doble clic en la entrada seleccionada
    o
  • Usa Ctrl + C
    y después Pega la contraseña en el campo deseado.
Nota: Usuario y contraseña también pueden copiarse al portapapeles con las opciones del menú Editar, aunque es mucho más trabajoso.
