关于主密码的话题

其中最重要的密匙，是为 Password Safe 数据库选择一个强有力的主密码保护您的密码（保险箱）安全。 在数据库（保险箱）中包含了你的所有用户名，密码，和可以添加的所有附加信息。记住最好不要选择这些密码，例如: 挑选配偶的姓名，生日，或其他一些容易推测的密码组合及您的敏感数据。重要的是，需要选择一个他人难以猜测但自己却能够轻松记住的主密码。

Password Safe 有一个内置的检查它认为薄弱主密码的功能。总而言之，一个主密码(无论你喜欢哪个密码) 应该由混合大小写字符和至少一个数字或特殊字符 （例如那些建立在键盘数字键上方的符号）。如果一个主密码输入程序被认为强度太弱，它将提示你确认。你可以选择接受刚输入的（弱强度）密码或重新输入新组合密码。

需要更改主密码，请查看更改主密码。

注意: 关于这点是没有办法在 Password Safe 里找回丢失或被遗忘的保险箱密码的！
这是一个有意设计的决定，因为使用任何机制来检索密码丢失造成的安全漏洞都可能被利用。
