创建与打开一个保险箱

Password Safe 允许用户存储所有密码在一个单独的 “保险箱”(密码数据库)或创建多个不同用途的数据库(例如: 一个为工作，一个为个人使用)。每个数据库能被移动和用于不同的系统上，只要安装了相同版本的 Password Safe。 数据库加密使用一个源于主密码的密匙(主密码无论如何不以任何形式存放在数据库中)。

如果是首次使用 Password Safe，可以单击新建数据库按钮，接着将询问用户选择密码数据库名称和位置(默认 "pwsafe.psafe3" 不错)。 然后，提示输入一个用于加密的主密码和锁定新保险箱内容。

创建一个新数据库

有两个途径可以创建一个新数据库。一个是当程序启动时从密码保险箱项目对话框单击新建数据库；另一个是在已打开的 Password Safe 从文件 > 新建...菜单。不管利用何种方法只要能促使密码保险箱设置对话框呈现便可。

备注: 默认时 Password Safe 将显示最后使用过的数据库， 和为它显示一个保险箱提示。如果使用多个数据库，可以在它们之间的下拉列表交替选择。 也可以输入路径或从文件对话框通过单击省略号(“...”)按钮选择打开数据库。 数据库一旦被打开，便可以从文件菜单通过打开数据库命令更改打开其它数据库。



虚拟键盘

要设置一个可靠的性的主密码，你可以使用字符（如：在使用德国键盘）像 ä, ö 一样或模拟。要实现如此字符的输入，你也许需要一个外国的键盘。这样的一个键盘就只能选择虚拟键盘了。该键盘一直以来被用于模拟所有可见的外国字符。虚拟键盘被显示为 标志。那些未见于物理键盘的外国字符，都将被显示在此。所以你可以创建你想要的主密码，但只有主密码可通过该方法创建。
