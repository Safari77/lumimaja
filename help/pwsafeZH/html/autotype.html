自动输入

自动输入为用户提供一个简易快捷输入用户名与密码信息的途径。 单击需输入的用户名和密码及递送主机认可的数据。默认状态，公式跟随自动击键如下: 用户名，接着换行、密码、再换行、最后输入键(用户名 换行密码 换行输入)。 该操作能满足大部分网站所需要的用户名/密码验证。

为了网站或应用程序需要不同的数据，可以在自定义自动输入章节下设计描述自动输入特性。注意也可以通过选项对话框的杂项选项卡自定义默认自动输入文本。 这是很有帮助的，如自动输入能非常快速的输入到计算机中。

使用自动输入

  1. 把光标置于表单的用户名字段里
  2. 最大化或打开 Password Safe
  3. 选定一个密码项目
  4. Ctrl -T 或右击和选定执行自动输入或在工具栏按钮单击启动。

自定义自动输入

由于网站或应用程序存在需要用户名字段和密码字段来使用登录按钮登录，可以修改“自动输入”字段特征指定想要填写的文本，特征如下:

  • \u 代替用户名关联项目。 
  • \p 代替密码关联项目。
  • \g 代替组关联项目。
  • \i  代替标题关联项目。
  • \o 代替备注 (可能多行文本) 关联字段。
    •  所有行字符“\n”、“\r”或“\r\n”结束均转换为“\r”(回车)。
    • 不代替生成转义序列在备注字段里 (例如: “\u”、“\p” 等。)除“\t”之外序列使用制表符取代。
  • \oNNN 是代替备注字段某行内容
    • NNN 可通过一至三个数字 (例如: \o5 为第五行 ，\o45 为第四十五行等等。)
    • 行尾字符不包括。
    • 如果 NNN 大于备注字段行数，则没有效果。
    备注: 这是一个最简便支持应用程序自动输入的方式，需要第二次密码或其它任意文本为认证过程部份。
  • \b 代替退格键。 
  • \t代替换行键，空指令字段有效。 
  • \n代替回车键，按回车按钮效果。 
  • \\代替反斜杠 ( \ ) 键。 
  • \dNNN 设置字符之间的延迟 NNN 毫秒， 而不用 10 (默认)。
    备注: 可以使用 \d 命令连贯一个数字。例如: \d999\d999\d999\d010 给予 3 秒延迟(3.007, 精确的)。 \d 命令设置字符间延迟和同时立即产生延迟。
  • 所有其它文本输入示列。

示例: 下列可用于银行帐号位置 (12345)，必须输入在用户和密码前:

12345\t\u\t\p\t\n

备注: 默认情况，自动输入字段不显示在添加或编辑对话框。 要访问自动输入字段，可以在对话框单击“更多>>”按钮。

如果自动输入存在问题，可能需要禁用第三方插件弹出拦截。即用户使用自动输入功能时弹出插件拦截报告问题的相互影响。
