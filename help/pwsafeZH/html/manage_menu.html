管理菜单

管理菜单是 Password Safe 的全部选项和控制设置。

更改保险箱密码

选择该选项弹出更改保险箱密码对话。
查看更改主密码的更多信息。

生成备份

创建当前密码数据库的一个备份副本。
查看备份与恢复数据库的更多信息。

恢复备份

恢复一个备份的密码数据库文件。
查看备份与恢复数据库的更多信息。

选项...

选择选项...弹出一个 Password Safe 具有多个设置可查看与编辑的选项卡对话框。要了解更多关于具体选项卡选择如下:

备份选项卡
显示选项卡
杂项选项卡
密码策略选项卡
历史密码选项卡
安全选项卡
系统选项卡
快捷方式选项卡
备注:
设置保存更多参数选择描述单击这里。


自定义工具栏

选定哪些图标需要显示于工具栏中和哪些则需要被隐藏。

生成密码

该菜单允许你依照密码策略，单击“生成”按钮生成的一组密码，并自动输入到剪贴板里。剪贴板中的密码可以在其后粘贴至其它某些应用程序中。
