使用已存储的密码

Password Safe 提供几种机制利用存储的用户名和密码。 大多需要复制用户名或密码到剪贴板，然后粘贴信息到所需输入的区域里。 Password Safe 提供一个自动输入功能，可以自动输入用户名和密码到一个网页表单里。

  1. 打开 Password Safe
  2. 选择希望使用的用户名/密码项目
    用户名:
  • 从工具栏选择用户名图标
    或
  • 右击和选择复制用户名到剪贴板
    或
  • 使用 Ctrl + U
    然后粘贴用户名到想要的区域里。
    密码:
  • 从工具栏选择密码图标
    或
  • 右击和选择复制密码到剪贴板
    或
  • 右击选定项目
    或
  • 使用 Ctrl + C
    然后粘贴密码到想要的区域里。
备注: 用户名和密码也可以通过选择设在编辑菜单被复制到剪贴板，但是坦率的说，点击它太远了点！
