编辑菜单

项目 组

通过编辑菜单可以添加、编辑、使用与组织用户名称及密码。 注意该菜单内容更改根据查看所选(例如: 选定项目或一个组)。并且，某些项目可以被禁用，描述如下。

添加项目

添加一个新用户名和密码。如果选择打开数据库为只读模式该项将被禁用。
查看用户名、密码与组获得更多信息。

编辑项目/展开/收缩组

当一个项目被选中时，编辑选中用户名与密码。 当一个组被选中时(在树状目录视图)，将展开或收缩项目显示在选中组之下。如果选择打开数据库为只读模式该项将被禁用。
查看用户名、密码与组获得更多信息。

删除项目/删除组

当一个项目被选中，该菜单将删除它。在选中一个组时 (在树状目录视图)，将删除所有选中组之下的项目。如果选中的组多于一个项目时则要求确认。如果选择打开数据库为只读模式该项将被禁用。

重命名项目/重命名组

重命名选中项目(或组，在树状目录视图)。如果选择打开数据库为只读模式该项将被禁用。

查找项目

查找一个包含特定文字的项目。默认状态，可查找除密码字段外的所有字段。
查看查找获得更多信息。

复制项目

复制选中项目。如果选择打开数据库为只读模式该项将被禁用。

添加组

创建一个新密码组(例如: 银行密码)。如果选择打开数据库为只读模式该项将被禁用。

清空剪贴板

清除剪贴板内存中记忆的所有内容。该项将清空剪贴板中被粘贴的任何 Password Safe 信息(密码、用户名、备注内容等)。 如果当前剪贴板没有包含来自 Password Safe 的内容，则该项操作不生效。

复制密码到剪贴板

复制当前选定的密码到剪贴板。

显示密码子集

该操作显示一个窗口允许指定哪些密码的字符被显示。这有助于站点要求登录密码组件时。 例如，一个密码可能为 "qAdm3ddENc" 和可能被询问输入第一、第三、第五和第八个字母。 该窗口允许制定一个合适方式: 在上面区域输入密码字符给密码子集。可以使用空格、逗号或分号分隔位置。 相应密码字符被显示在下面的区域中。需要退出，请使用 Esc 键。

复制用户名到剪贴板

复制当前选定的用户名到剪贴板。

复制备注到剪贴板

复制当前选定的备注字段内容到剪贴板。

复制网址到剪贴板

复制当前选定的网址字段到剪贴板。

复制邮箱到剪贴板

复制当前选中的邮箱字段到剪贴板。

复制运行命令到剪贴板

复制当前选中的运行命令字段到剪贴板。

浏览网址

当选中一个项目，并且该项目具有相应网址关联，则该项将调用默认浏览器浏览此网址。 如果未选中项目或选中的项目网址为空白，该命令将被屏蔽。

浏览网址 + 自动输入

当一个项目被选中和该项目存在其相关的网址时，则该项目将调用默认浏览的网址。 如果未选择项目或所选项目网址为空，该命令将被禁用。

自动输入用户名和密码到所选的输入字段里。
查看 使用自动输入 获取更多信息。

发送邮件

准备发送当前选定的电子邮件字段中的邮件数据。
查看 用户名称、密码和组 获取更多信息。

执行自动输入

自动输入用户名与密码输入到选定的区域里。
查看 使用自动输入 获得更多信息。

运行命令

该操作将执行所选项目字段 运行命令 中的命令。
查看 使用运行命令 获取更多信息。

创建快捷方式

当一个项目被选中，该项允许创建一个引用给项目代表其它组。这有助于假如有一个项目 “应被放在”超出一个组之下，但恰好又不希望复制数据时。一个快捷方式具有一个截然不同的图标 相关联与存在默认名称 “快捷方式到 x”， 在 x 位置为原项目标题。当然，你可以编辑名称与移动或删除快捷方式，只要是符号规定的项目。

转到基本项目

该项将只显现在一个快捷方式或别名项目。 在单击转到基本项目之后对应的基本项目将被选中。

鼠标右击操作

许多功能通过在相应项目或组上右击鼠标按钮，在弹出的编辑菜单上同样可以实现。
