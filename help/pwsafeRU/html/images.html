Значки элементов

Ниже приведены значки, используемые для отображения типов элементов в древовидном и простом списках. Примечание. В древовидном списке значки отображаются всегда. В простом списке столбец со значками может быть добавлен только как первый столбец.

 Группа [только в древовидном списке]

 Пустая группа [только в древовидном списке]

 Обычный элемент

 Обычный элемент с псевдонимами

 Обычный элемент с ярлыками

 Псевдоним

 Ярлык

Кроме того, у обычных элементов может быть задана дата истечения пароля. Если для таких элементов настроено предупреждение (например, за 5 дней до истечения), их значки станут наполовину пурпурными при наступлении срока предупреждения.

 Обычный элемент — срок действия пароля истечёт в течение указанного в предупреждении времени.

 Обычный элемент с псевдонимами — срок действия пароля истечёт в течение указанного в предупреждении времени.

 Обычный элемент с ярлыками — срок действия пароля истечёт в течение указанного в предупреждении времени.

У элементов с истёкшим паролем весь значок становится пурпурным.

 Обычный элемент — срок действия пароля истёк.

 Обычный элемент с псевдонимам — срок действия пароля истёк.

 Обычный элемент с ярлыками — срок действия пароля истёк.
