Menü Datei

Das Menü Datei enthält die Punkte, um Password Safe Datenbanken erstellen und bearbeiten zu können.

Neu

Erstellt eine neue leere Password Safe Datenbank.

Öffnen

Öffnet eine bereits bestehende Password Safe Datenbank.

Schließen

Schließt die aktuell geöffnete Password Safe Datenbank, ohne die Anwendung zu verlassen.

Zuletzt

Dieser Teil im Menü Datei enthält die Liste der zuletzt geöffneten Datenbanken. Diese Liste ist leer beim ersten Einsatz von Password Safe.

Hinweis:
  • Die Liste der zuletzt geöffneten Dateien kann gelöscht werden, wenn Sie "Aktuelle Liste löschen" auswählen.
  • Die Anzahl der zuletzt geöffneten Dateien die sich Password Safe merkt kann mit dem Dialog Verwalten->Optionen->System eingestellt werden. Hier können Sie auch festlegen ob Sie die Liste im Menü Datei sehen wollen, oder lieber in einem Untermenü.

Speichern

Speichert jede Änderung zur aktuellen Datei ab. Wenn seit der letzten Speicherung keine Änderungen gemacht wurden, oder wenn die Datenbank im NUR Lesen Modus geöffnet wurde, ist dieser Punkt ausgegraut.

Speichern unter

Speichert eine Kopie der aktuellen Datenbank mit neuem Namen und Speicherort ab.

Exportiere zu

Exportiert den Inhalt der aktuellen Datenbank zu einer externen Datei in einem früheren Password Safe Datenbankformat, in XML, oder als Text ab. Siehe hier für mehr Information wie man Datenbanken exportiert in anderen Formaten.

Importiere von

Importiert den Inhalt einer früher exportierten oder korrekt formatierten Textdatei, XML, oder eine KeePass Passwort Datenbank in die aktuelle Datenbank. Siehe hier für mehr Information wie man Textdateien importiert.

Zusammenführen

Führt eine Password Safe Datenbank oder ein Sicherungskopie (.bak) zusammen mit der aktuellen Datenbank. Dies ist im NUR Lesen Modus gesperrt.

Vergleichen

Vergleicht die aktuell geöffnete Datenbank mit einer anderen Password Safe Datenbank.

Synchronisieren

Synchronisiert die aktuell geöffnete Datenbank mit einer anderen Password Safe Datenbank. Dies ist im NUR Lesen Modus gesperrt.

Der Unterschied zwischen Zusammenführen und Synchronieren besteht in der Tatsache das beim Zusammenführen sämtliche Elemente aus der synchonisierenden Password Safe Datenbank in die aktuell geöffneten Datenbank übernommen werden, beim Synchronisieren werden nur die in der aktuell geöffneten Datenbank vorhandenen Elemente abgearbeitet.

Hinweis: Die Punkte Zusammenführen, Vergleichen und Synchronisieren können einige Zeit in Anspruch nehmen, vor allem bei große Datenbanken.

Zu R-O ändern

Die aktuell geöffnete Datenbank ist entweder im Modus R-O (read only = nur Lesen) oder Modus R/W (read / write = Lesen / Schreiben). Zwischen diese beiden Modi kann umgeschaltet werden.

Hinweis: Ein in Modus R-O geöffnete Datenbank kann nur dann in Modus R/W umgeschaltet werden, solange keine Änderung vorgenommen wurde.

Eigenschaften

Zeigt Informationen zur aktuell geöffneten Password Safe Datenbank an, wie da sind: Format, Version, Anzahl Gruppen, Anzahl Einträge, wer was zuletzt gespeichert hat. Der Name der Datenbank und eine Beschreibung derselben können bearbeitet werden.

Beenden

Beendet die Password Safe Anwendung. Wenn seit der letzten Speicherung noch Änderungen vorgenommen wurden, werden Sie aufgefordert die Datenbank abzuspeichern.

Berichte

Hinweis: Die Aktionen Vergleichen, Exportieren, Inportieren, Zusammenführen und Synchronisieren erzeugen alle einen Bericht. Dieser Bericht kann eingesehen werden, und dann in die Zwischenablage oder als Datei kopiert werden als Textdatei im selben Verzeichnis wie die Datenbank. Das Menü "Ansicht → Berichte" kann dazu verwendet werden diese Berichte zu einem späteren Zeitpunkt anzuschauen. Der Name der Berichtsdatei ist fest eingestellt und abhängig von der Funktion die den Bericht erzeugt hat. Wenn dieselbe Funktion wieder ausgeführt wird, und der Bericht wird als Datei abgespeichert, wird der vorherige Bericht unweigerlich überschrieben. Hinweis: Der Bericht kann empfindliche Daten enthalten, und darum ist es ratsam die Funktion Auf Festplatte Speichern mit Bedacht auszuführen.
