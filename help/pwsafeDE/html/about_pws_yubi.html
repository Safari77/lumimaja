Password Safe und YubiKey

YubiKey ist ein innovatives Authentifizierungsgerät in Hardware eingebettet, entwickelt und vertrieben von Yubico. Für den Anwender, sieht es aus wie ein kleine, dünne USB-Adapter mit einer Taste. In Password Safe funktioniert es wie ein Authentifizierungs Token, der eine große hardwaremäßige Sicherheit dartsellt für die Datenbank.

YubiKey beschaffen

Zum Anfangen braucht man zuerst mal ein YubiKey Gerät. Diese können online hier beschafft werden. Achten sie darauf dass Password Safe im YubiKey eine Firmwareversion von mindestens 2.2 voraussetzt.

Konfiguration des YubiKey

Der YubiKey muss richtig konfiguriert werden, um den Betrieb mit Password Safe zu gewährleisten. Dies kann gemacht werden im Verwalten->Yubikey... Dialogfenster, wie beschrieben im Menu Verwalten.
Wenn Sie schon ein YubiKey haben, achten Sie darauf dass Password Safe Slot 2 für die Authentifizierung verwendet, und dabei Slot 1 für andere Anwendungen frei lässt.

Verwendung von YubiKey

Einaml konfiguriert, können Sie es einsetzen um zusammen mit dem Master-Passwort die Password Safe Datenbank zu schützen, wie beschrieben in Neue Datenbank erstellen.
