User Names, Passwords, and Groups: Password Policy

Username, passwords and groups together with the additional parameters are managed in 4 tabs:

  • Basic: all necessary elements for password management
  • Additional: additional specifications, including history
  • Dates and Times: expiry and statistics
  • Password Policy: specific differences to the default

Password Policy Tab


If you don't want to use the default password policy (see Setting a Password Policy), you can define in this tab a different password policy, marking "Use the policy below" and setting the appropriate elements. You may also select your own set of symbols to use when generating a password contaning them. You may also set this list for the whole database via "Manage → Options → Password Policy" tab and override it for specific entries.
